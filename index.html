Названия кманд Bash:

  • Названия команд
    • mkdir
      • Процес создания папки.
        • mkdir "название файла"
    • rmkdir
      • Процесс удаления папкиюю.
        • rmkdir "название файла"
    • cd
      • Вход в папку.
        • cd "название файла"
    • cd ../
      • Выход из папки.
        • cd../ "название файла"
    • touch
      • Создания файла "txt."
        • touch "название файла"."его формат"
    • code
      • Процесс ооткрывания кода обекта
        • code "название файла"."его формат"
    • rm
      • Процесс удаления файла
        • rm "название фаила"
    • ls
      • Проверка что находидтся в обекте
        • ls
    • git init
      • Процесс удаления папкию
        • git init
